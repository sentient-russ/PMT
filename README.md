### Overview:
Project management prototype created for Software Project Management SWE SWE4633 Spring 2024.

### High-level technologies:
> Java, Netbeans, MySql

### Requirements-Based Features:
>- Prominently displayed list of all projects with expended hours
>- Add/Changeproject function
>- Add/Update project description function
>- Add/Update project owner name function
>- Create/Update/Delete project team member function
>- General overview screen for each project
>- Create/Update/Delete project description function
>- Create/Update/Delete product owner function
>- Create/Update/Delete team members function
>- List of all risks with their status
>- Create/Update/Delete risks function
>- Create/Update/Delete functional requirements
>- Create/Update/Delete non-functional requirements
>- Displays total expended hours
>- Daily effort input functions
>- Daily effort Add/Change function for Requirements Analysis hours
>- Daily effort Add/Change function for Designing hours
>- Daily effort Add/Change function for Coding hours
>- Daily effort Add/Change function for Testing hours
>- Daily effort Add/Change function for Project Management hours
